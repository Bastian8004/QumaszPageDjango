<h1>Qumasz</h1>

<p>Qumasz to strona zrobiona na zlecenie dla firmy od korepetycji z matematyki, która pozwala firmie na dodawanie opinii z google oraz prowadzenie bloga.</p>

<h2>Funkcjonalności</h2>
<ul>
    <li>Dodawanie opinii z google</li>
    <li>Prowadzenie bloga</li>
</ul>

<h2>Technologie</h2>
<ul>
    <li>Django 3.x</li>
    <li>Python 3.x</li>
</ul>
